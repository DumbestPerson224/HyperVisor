About us

This app is aimed at providing a free and open source virtualization software.

Privacy Policy

This platform collects data such as Virtual Machine errors, accounts and Virtual Machine settings.

This platform will never collect personal information such as names, emails (except for parental controls, passwords, etc

Virtual machine settings saved include:

  • Wifi related stuff
  • Boot configuration
  • OS installed
  • Disk space
  • Partitioning
  • Memory settings

Parental Controls data collection

This will send all data to a parent's email once sent all unessential data is deleted from the server

With this a parent can:

  • Track search history
  • Apply custom restrictions and passwords
  • Get everything a child is doing
